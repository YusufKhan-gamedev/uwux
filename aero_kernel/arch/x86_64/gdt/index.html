Module aero_kernel::arch::x86_64::gdt

source · [−]
Expand description

The GDT contains entries telling the CPU about memory segments.

In our case we only need a descriptor table in x86_64 (as in every other arch that Aero currently support) as those arch’s do not implement segmentation at all.
